# Honey Gold Store

Premium skincare and wellness e-commerce platform built with React, TypeScript, and Tailwind CSS.

## Project Overview

ADEDAS MULTIBUSINESS LTD offers premium skincare products including body creams, shea butter, essential oils, and wellness essentials.

## Tech Stack

- **Frontend**: React 18, TypeScript, Vite
- **Styling**: Tailwind CSS, shadcn-ui
- **State Management**: React Context (Cart), React Query
- **Routing**: React Router DOM
- **Forms**: React Hook Form, Zod
- **Charts**: Recharts

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Features

- Product catalog with categories
- Shopping cart functionality
- Admin dashboard
- Product management
- Customer testimonials
- WhatsApp contact integration
